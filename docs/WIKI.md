# GitHub Wiki

The GitHub Wiki content is maintained in `docs/wiki/` so changes can be reviewed in pull requests.

To publish or refresh the GitHub Wiki:

1. Make sure the repository Wiki feature is enabled.
2. Open the repository Wiki in GitHub once if the `.wiki.git` remote has not been initialized yet.
3. Clone `https://github.com/aditaa/CraftLedger-Jobs.wiki.git`.
4. Copy the Markdown files from `docs/wiki/` into the wiki clone.
5. Commit and push the wiki repository.

The wiki pages currently cover installation, commands, configuration, testing, volunteer beta testing, permissions, and troubleshooting.
