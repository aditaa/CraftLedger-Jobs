# Testing

CraftLedger Jobs beta testing should start from a downloaded beta release jar.

Use the [Volunteer Beta Testing Guide](VOLUNTEER_TESTING.md) for player testing. It explains how to:

- download the beta release for your Minecraft version
- install the mod in a Forge client with the vanilla Minecraft Launcher
- install the mod in a CurseForge profile
- start a fresh singleplayer test world
- run command, shop, sell, job, reward, operator, and optional config tests
- report bugs and successful tests with screenshots

Maintainers can track release readiness in the [Release Checklist](RELEASE_CHECKLIST.md).
