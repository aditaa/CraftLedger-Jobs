# GitHub Setup

Repository settings that are not fully represented by files.

## Social Preview

Use `.github/assets/social-preview.png` as the repository social preview image.

GitHub path:

1. Open repository settings.
2. Go to General.
3. Find Social preview.
4. Upload `.github/assets/social-preview.png`.

## Discussions Categories

Discussions are enabled. Recommended categories:

- `Announcements`: release notes, staging updates, and maintainer-only news
- `General`: broad project conversation
- `Ideas`: feature proposals and design sketches
- `Q&A`: user and server-owner questions
- `Support`: install, config, and troubleshooting help
- `Testing`: staging-server reports and compatibility checks
- `Server Admins`: economy tuning, backups, permissions, and operations

GitHub currently exposes discussion category readback through APIs, but category creation is a repository settings UI task.
