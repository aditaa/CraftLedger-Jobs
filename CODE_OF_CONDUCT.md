# Code of Conduct

CraftLedger Jobs should be a welcoming project for server owners, mod developers, contributors, and players.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, especially with new contributors.
- Keep feedback focused on the code, docs, tests, or behavior.
- Help make the project easier to understand.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language
- Publishing private information
- Deliberately disruptive comments or pull requests
- Abusive behavior toward maintainers, contributors, or users

## Enforcement

Maintainers may edit, hide, or remove comments and may block contributors who repeatedly violate this code of conduct.
